# homelab_sop

这是一个记录搭建和维护家庭实验室（homelab）各类硬件与软件操作规范、指南与笔记的仓库。

## 概述

本仓库记录了 homelab 项目的实用部署方案、配置步骤和故障排查技巧。目标是帮助保存经验，降低重复劳动，并让不同设备与软件平台之间的操作可复用。

## 目录结构

- `docs/` - 详细的指南和操作文档。
  - `极空间 + Linux VPS 存储映射终极指南.md` - 关于极空间或类似存储系统与 Linux VPS 之间存储映射的实战指南。
- `src/` - 可放置用于支持文档的脚本、自动化代码或配置示例。

## 使用说明

1. 查阅 `docs/` 目录中的文档，选择对应的操作指南。
2. 按照文档中的步骤逐步执行配置与部署。
3. 随着 homelab 的发展，补充新的笔记、脚本或操作流程。

## 贡献指南

- 欢迎添加新的文档或完善已有指南。
- 保持内容简洁、实用、可复现。
- 使用明确的章节标题，并说明必要的前置条件或假设。
